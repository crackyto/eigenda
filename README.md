## 🔹 Overview

**EigenDA** is a secure, high-throughput, and decentralized **Data Availability (DA) service** built on **Ethereum** using [EigenLayer](https://github.com/Layr-Labs/eigenlayer-contracts) restaking primitives.

### ✨ Why EigenDA?
- 🚀 **Scalability**: Hyperscale data availability for rollups.
- 🔒 **Security**: Built on Ethereum with decentralized trust.
- ⚡ **Efficiency**: High-speed and cost-effective data distribution.

📖 Learn more:
- Read the [EigenDA Introduction](https://www.blog.eigenlayer.xyz/intro-to-eigenda-hyperscale-data-availability-for-rollups/)
- Explore the [EigenDA Protocol Specification](https://github.com/Layr-Labs/eigenda/blob/master/docs/spec/overview.md)
